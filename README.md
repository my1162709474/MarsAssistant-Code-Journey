**文件**: `scripts/20260204_078_color_theme_generator.py` ⭐ **最新**

**名称**: Color Theme Generator - 终端颜色主题生成器与预览工具

**功能**:
- 🔧 **完整颜色处理**: Color类支持HEX/RGB/HSL/ANSI 256/真彩色
- 📊 **高级颜色分析**: 对比度计算(符合WCAG标准)、亮度计算、颜色混合
- 🎨 **流行预设主题**: Nord、Dracula、Catppuccin、Solarized
- 📦 **动态主题生成**: 单色系、类似色、补色、三色主题
- 🧠 **多格式导出**: JSON、CSS变量、Tailwind配置、Python字典
- 🖥️ **终端预览**: ANSI真彩色实时预览效果

**核心类**:
- `Color` - 完整的颜色处理类
- `ColorTheme` - 颜色主题管理与导出
- `ColorThemeGenerator` - 主题生成器（静态方法）

**使用示例**:
```python
# 生成并预览主题
python color_theme_generator.py --demo

# 预览特定主题
python color_theme_generator.py --preview nord

# 导出主题为多种格式
python color_theme_generator.py --preview random-dark
# 输入 y 确认导出
```

# AI提示工程技术示例
## Day 12: 高级提示模式与最佳实践

本文件展示了多种经过验证的AI提示工程技术，帮助开发者更有效地与大型语言模型交互。

---

## 1. 零样本提示 (Zero-shot Prompting)

直接提问，不提供示例。

```python
zero_shot_prompt = """
请将以下英文句子翻译成中文：
"The only way to do great work is to love what you do."
"""
```

---

## 2. 少样本提示 (Few-shot Prompting)

提供几个示例来引导模型理解任务。

```python
few_shot_prompt = """
任务：将形容词转换为副词

示例：
- "quick" → "quickly"
- "happy" → "happily"
- "careful" → "carefully"

请转换："intelligent"
"""
```

---

## 3. 思维链提示 (Chain-of-Thought, CoT)

引导模型逐步思考复杂问题。

```python
cot_prompt = """
问题：如果一根绳子长10米，我每天用掉2米，几天后绳子会用完？请一步步推理。

推理过程：
1. 绳子初始长度：10米
2. 每天用掉：2米
3. 用完需要的天数 = 总长度 ÷ 每天用量
4. 10 ÷ 2 = 5天

答案：5天
"""
```

---

## 4. 自洽性提示 (Self-Consistency)

生成多个答案，通过投票选择最一致的。

```python
def self_consistency_prompt(question, num_samples=3):
    """生成多个推理路径，取多数一致的答案"""
    return f"""
    问题：{question}
    
    请生成{num_samples}种不同的解答方案，然后选择最合理的答案。
    """
```

## 5. 角色设定提示 (Role Prompting)

给AI分配特定角色以获得更专业的回答。

```python
role_prompt = """
你是一位经验丰富的软件架构师，拥有20年的大型系统设计经验。

请为以下需求设计系统架构：
一个需要支持100万并发的实时聊天应用。
"""
```

---

## 6. 结构化输出提示

要求模型按特定格式输出，便于程序解析。

```python
structured_prompt = """
请分析以下代码的性能问题，并按JSON格式输出：

{
  "问题列表": [
    {
      "行号": "整数",
      "问题": "问题描述",
      "严重程度": "高/中/低",
      "优化建议": "建议方案"
    }
  ],
  "总体评分": "1-10的评分",
  "关键改进点": ["最重要的3个改进点"]
}

代码：
{user_code}
"""
```

## 7. 迭代优化提示

通过多轮对话逐步完善结果。

```python
iterative_prompt_v1 = """
请写一个Python函数，实现冒泡排序。
"""

iterative_prompt_v2 = """
上一版本的代码缺少：
1. 类型注解
2. 文档字符串
3. 错误处理

请改进上述代码。
"""

iterative_prompt_v3 = """
现在请添加：
1. 时间复杂度分析
2. 单元测试用例
3. 使用示例
"""
```

## 8. 约束提示 (Constraint Prompting)

设定明确的边界条件。

```python
constraint_prompt = """
请用Python实现一个函数，满足以下约束：
1. 不使用任何内置排序函数
2. 时间复杂度必须是 O(n log n)
3. 函数名必须是 "efficient_sort"
4. 输入必须是整数列表
5. 必须返回一个新列表（原地修改）
6. 函数不超过50行代码
"""
```

## 9. 上下文学习提示 (In-Context Learning)

提供详细背景信息帮助模型理解任务。

```python
icl_prompt = """
【背景知识】
我们的产品是一款面向中国市场的AI助手，主要用户是大学生和职场新人。

【用户画像】

【任务】
请为这款AI助手设计5个实用的Prompt模板，用于论文写作辅助。
"""
```

## 10. 元提示 (Meta-Prompting)

让AI自己生成和优化提示。

```python
meta_prompt = """
我需要写一个提示词来让AI帮我：
- 分析用户反馈的情感倾向
- 提取关键问题点
- 生成回复建议

请帮我设计一个结构化的提示词模板，要求：
1. 包含情感分类的选项
2. 问题提取的结构化格式
3. 回复建议的评估标准
"""
```

## 最佳实践总结

| 实践 | 说明 | 适用场景 |
|------|------|----------|
| 清晰具体 | 明确说明期望的输出格式和长度 | 所有场景 |
| 分步指令 | 复杂任务拆分成多个步骤 | 复杂问题 |
| 提供示例 | 2-5个典型示例效果最佳 | 任务不明确时 |
| 设定角色 | 让AI扮演专家角色 | 需要专业回答时 |
| 迭代优化 | 通过多轮对话完善结果 | 高质量输出需求 |
| 明确约束 | 列出边界条件和限制 | 需要精确控制时 |
